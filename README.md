# Overview
This project introduces the Occurrence Birth-Death Diffusion model and studies the diversification dynamics of cetaceans.

Some files are too large for this repository: trees from https://academic.oup.com/sysbio/article/70/5/922/6122448, model outputs that can be regenerated with the Julia scripts.

# Directory Structure and Description
- `Calibration`: Validating the OBDD model.
- `Cetacea`: Diversification analyses.
